# **Pre-requisite:**
  1. Download [Github Desktop](https://desktop.github.com/download/)
  2. Download [Git Large File Storage](https://git-lfs.com)


## **Fork this Repo First!!!**

### To know if it's working:
1. Open **Github Desktop**
2. Click Clone Repository or Press Ctrl+Shift+O
3. Then Copy your forked repo and paste it there
> Optional: Choose a path to your desired location/directory *(Mine is in /Documents/Code/Unity/Game-Development)*


4. Open **Unity Hub**
5. Click "Add" > "Add project from disk"
6. Click the **Game-Development** from Project
7. Then happy coding!
###### lala


### To update your fork repo:
1. Open Github Desktop
2. You can see the changes and history
3. Fill the Summary text field to commit
> Optional: Add descriptions to know what's the update on your end

4. **Push** in your fork
5. **ALWAYS** sync/pull your latest commit
> There can be error the next time you commit after your push
